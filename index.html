Introduction

This Project deal with some real life problems and outcome with good accuracy. This problem solved using Machine Learning Python.
For Learning model I firsly load the datasets (i.e. CSV ) and appling preprocessing methods which invoved in few steps:
(i) Finding Null value.
(ii) Replace null values with mode,medium,mean as per model requirements
Then divide the datasets for training and testing purpose (eg.80% for training and 20% for testing).Then apply the appropriate algorithmns and predict the results.
After model is ready its time to deployment so I use Flask because its very lite weighted and directly interact with Machine Learning libraries and HTML and CSS/JS.

Affecting Factors of Machine Learning?

There are few factor which affect the accuracy of Machine Leartning:

(iii) Selection of features in the data:Selection of independent indexs
